## Que isso?

Traduções **pt-br** de jogos desktop e atalhos de terminal para listar, baixar e ler tutoriais.

Site: [tradu.pages.dev](https://tradu.pages.dev/)

## Atalhos
```bash
curl -sSL tradu.pages.dev/install.sh | bash      # ativar
curl -sSL tradu.pages.dev/uninstall.sh | bash    # remover
```
| Atalho | Comando |
| :--- | :--- |
| Lista as traduções | `tradu` |
| Baixa a tradução direto | `tradu -d [nome-do-jogo]` |
| Mostra os tutoriais | `tradu -t [nome-do-jogo]` |
| Abre a tela de ajuda | `tradu -h` |
